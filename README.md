## 📖 Sobre o projeto

Este repositório documenta minha jornada no desafio de **30 dias de JavaScript**, com o objetivo de fortalecer a lógica de programação e consolidar os principais conceitos da linguagem.

Cada dia traz um novo desafio, explorando desde fundamentos básicos até aplicações mais interativas no navegador.

---

## ✨ O que estou praticando

- Manipulação do DOM  
- Eventos  
- Funções  
- Arrays e métodos  
- Lógica de programação  
- LocalStorage  
- Integração com APIs  
- Boas práticas com JavaScript  

---

## 🛠️ Tecnologias utilizadas

- JavaScript (Vanilla JS)  
- Git & GitHub  

---

## 📅 Estrutura do desafio
